Самая быстрая в мире платформа для создания веб-сайтов

Hugo - один из самых популярных генераторов статических сайтов с открытым исходным кодом. Благодаря своей удивительной скорости и гибкости, Hugo снова делает создание веб-сайтов увлекательным.

Пылающая скорость

Что современного в ожидании создания Вашего сайта?

Hugo - самый быстрый инструмент в своем роде. При <1мс на страницу в среднем сайт создается менее чем за секунду.

Надежное управление контентом

Правила гибкости. Hugo - мечта стратега по контенту.

Hugo поддерживает неограниченное количество типов контента, таксономий, меню, динамического контента на основе API и многого другого, и все это без плагинов.

Шорткоды

Шорткоды Hugo - скрытая суперсила Markdown.

Нам нравится красивая простота markdown синтаксиса, но бывают моменты, когда нам нужна большая гибкость. Шорткоды Hugo сочетают в себе красоту и гибкость.

Встроенные шаблоны

Hugo имеет общие шаблоны для быстрого выполнения работы.

Hugo поставляется с готовыми шаблонами для быстрой работы с SEO, комментированием, аналитикой и другими функциями. Одна строка кода, и все готово.

Многоязычный и i18n

Полиглот.

Hugo обеспечивает полную поддержку i18n для многоязычных сайтов с тем же простым опытом разработки, который нравится пользователям Hugo на одноязычных сайтах.

Пользовательские выводы

HTML недостаточно?

Hugo позволяет выводить контент в нескольких форматах, включая JSON или AMP, и позволяет легко создавать свои собственные.
 

Спонсоры Hugo

300+ тем

Hugo предлагает надежную систему тематики, которую легко реализовать, но которая способна создавать даже самые сложные веб-сайты.

Ознакомьтесь с темами Hugo.

Возможность создания шаблонов

Шаблоны Hugo на основе Go предоставляют необходимое количество логики для создания чего угодно, от простого до сложного.

Установить за секунды, построить за миллисекунды.

Hugo работает на macOS, Windows, Linux, FreeBSD и других.

Хостинг на любом сервере или на Вашем любимом CDN.

Mac OS

$ brew install hugo

Windows

$ choco install hugo -confirm

Linux

$ snap install hugo

См. другие методы в Руководстве по началу работы.
